1 : 
Create namespace <name>
command : kubectl create namespace <namespace_name>

2 : 
List one or more namespaces
command : kubectl get namespace <namespace_name>

3 : 
Display the detailed state of one or more namespace
command : kubectl describe namespace <namespace_name>

4 : 
Delete a namespace
command : kubectl delete namespace <namespace_name>

5 : 
Edit and update the definition of a namespace
command : kubectl edit namespace <namespace_name>

6 : 
Display Resource (CPU/Memory/Storage) usage for a namespace
command : kubectl top pods -n <namespace_name>
