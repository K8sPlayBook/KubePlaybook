1 : 
List one or more daemonsets
command: kubectl get daemonset

2 : 
Edit and update the definition of one or more daemonset
command: kubectl edit daemonset <daemonset_name>

3 : 
Delete a daemonset
command: kubectl delete daemonset <daemonset_name>

4 : 
Create a new daemonset
command: kubectl create daemonset <daemonset_name>

5 : 
Manage the rollout of a daemonset
command: kubectl rollout daemonset

6 : 
Display the detailed state of daemonsets within a namespace
command: kubectl describe ds <daemonset_name> -n <namespace_name>
