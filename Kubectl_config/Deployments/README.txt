1 : 
List one or more deployments
command : kubectl get deployment

2 : 
Display the detailed state of one or more deployments
command : kubectl describe deployment <deployment_name>

3 : 
Edit and update the definition of one or more deployment on the server
command : kubectl edit deployment <deployment_name>

4 : 
Create one a new deployment
command : kubectl create deployment <deployment_name>

5 : 
Delete deployments
command : kubectl delete deployment <deployment_name>

6 : 
 See the rollout status of a deployment
command : kubectl rollout status deployment <deployment_name>
